文字过滤模板：有前导字符的正则表达式

前导字符：
	- : ignore，忽略该表达式（注释）
	= : common，正则表达式的第一个字符为非控制字符，后接正则表达式（如果使用'+'也不会有问题，但最好不要用错）
	+ : advance，正则表达式的第一个字符为控制字符，后接正则表达式（绝对不能用错为'='）
	@ : insensitive，正则表达式中的字母全部忽略大小写，后接'='或'+'，用法与上面类似，但要注意，如果正则表达式的第一个字符为字母，也要使用'+'而不是'='，一定要小心
注：由于'@='很容易用错，因此可以将'@='全部使用'@+'，弃用'@='
